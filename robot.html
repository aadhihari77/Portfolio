Knights Tour Mobile Robot

Project Overview

Collaborated in a team of five to develop a System Verilog-based design for solving the Knights Tour problem, enabling a robot to cover a chessboard while visiting all squares exactly once.

Key Responsibilities

  • Designed testbenches and integrated with Python scripts for simulation control and waveform analysis.
  • Managed log collection and file management to ensure efficient verification and debugging.
  • Synthesized the design for real-world implementation on an Altera Cyclone-IV FPGA.

Technologies Used

System Verilog Python FPGA Altera Cyclone-IV
